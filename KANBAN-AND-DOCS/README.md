# Clip4Clips — Kanban & Docs

> This is an **orphan branch** (`kanban-and-docs`) holding the business & vision
> docs for the Clip4Clips / CLIPTI Trident. The **code** lives on the `main` branch.

## The Trident (three prongs)

1. **ClipForge** — the software fleet / done-for-you clip factory. *The gate.*
   Cash + audience + pipeline proof. (Code on `main`.)
2. **ClipTi** — the tap-to-clip hardware: Ring + Band + NFC tags + Desktop Hub.
   *The product.* A physical capture button that bridges a real-world moment to a
   published clip.
3. **Ecosystem** — CLIPTI OS / creator network / marketplace. *The lock-in.*
